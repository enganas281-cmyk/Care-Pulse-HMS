# CarePulse Hospital Management System

A complete frontend-only hospital management system built with React, Vite, Tailwind CSS, React Router, Lucide icons, and static mock data.

## Run Locally

```bash
npm install
npm run dev
```

## Included Modules

- Dashboard
- Patients
- Appointments
- Doctors
- Departments
- Admissions
- Pharmacy
- Billing
- Laboratory
- Reports
- Settings

No backend, database, or API is required.
